# 🎬 Netflix Content Strategy Analysis & Portfolio Insights

## 📌 Project Overview
An exploratory data analysis (EDA) on Netflix's global catalog of 8,800+ movies and TV shows. This project analyzes content trends, production geographies, release timing, and genre distribution to derive data-driven recommendations for portfolio diversification and release scheduling.

## 🛠️ Tech Stack & Tools
- **Language:** Python
- **Data Manipulation:** Pandas, NumPy
- **Data Visualization:** Matplotlib, Seaborn

## 🔑 Key Findings
- **Geographic Production:** The United States, India, and the United Kingdom dominate total content output, highlighting core markets for international investment.
- **Release Timing:** Content additions peak significantly in **July and December** and heavily favor **Friday launches** to maximize weekend audience engagement.
- **Underrepresented Genres:** Niche categories such as *Stand-Up Comedy* and *Sci-Fi & Fantasy* represent growth opportunities for expanding audience reach.

## 💡 Strategic Recommendations
1. **Optimize Release Schedules:** Align major original content releases with peak viewing windows (Fridays and mid/end-of-year holiday periods).
2. **Diversify Genre Investments:** Increase production and licensing in underrepresented high-engagement genres to attract broader demographic segments.
3. **Expand Localized Production:** Continue capital investment in top international hubs (e.g., India and regional Europe) for localized content strategies.

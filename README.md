# Netflix Content Analysis

## Project Overview
This project analyzes Netflix content dataset to identify trends in genre distribution, country contribution, release patterns, and content type.

## Objective
To perform exploratory data analysis and generate business insights from Netflix dataset using Python.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis Performed
- Data cleaning and preprocessing
- Content distribution by country
- Genre popularity analysis
- Trend analysis over time
- Duration analysis
- Correlation analysis

## Key Insights
- Movies dominate over TV shows in overall content.
- USA contributes the highest number of titles.
- Significant growth in content addition post-2015.
- Certain genres consistently outperform others.

## Business Recommendations
- Increase focus on high-performing genres.
- Expand content production in emerging regions.
- Optimize duration formats based on audience trends.

## Project Structure
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Business Insights
